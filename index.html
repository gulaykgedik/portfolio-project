Projeler

X Clone

X-Clone is a social media application where users can log in with Google or email, create tweets, edit and delete them, and like tweets. 🛠 Technologies Used
* React - Component-based UI development
* Firebase - Authentication and data management
* TailwindCSS - Modern and stylish design
* Moment.js - Formatting timestamps
* React-Icons - UI icons
* React-Router-DOM - Page navigation
* React-Toastify - Toast notifications
* UUID - Generating unique tweet IDs

Chat App

This project is a Chat Application that allows real-time messaging using Firebase. You can log in with Google and chat in different rooms. It features a fun and modern design with a fully responsive layout. 🛠 Technologies Used
* React – Component-based structure
* Firebase – Real-time database and authentication
* TailwindCSS – Responsive and modern UI
* Emoji-Picker-React – Add emojis to messages
* React-Icons - UI icons
* React-DOM – Optimized virtual DOM performance

ShopTime

ShopTime is a modern e-commerce platform designed to enhance the shopping experience with its user-friendly interface, categorized products, cart management, and dark/light mode features. Features
* Global State Management using Redux Toolkit.
* Page Navigation with React Router DOM.
* Pagination using React Paginate.
* Responsive Design: Fully compatible with all devices.
* Category Management: Products are displayed in organized categories.
* Cart Functionality: Add/remove products, manage quantities, and calculate total prices.
* Stylish Sliders: Implemented with React Slick and Slick Carousel.

YouTube Clone

This project is a web application that replicates the core features of YouTube. It offers a modern design and user-friendly interface, allowing users to search for videos, watch them, and view their details. Features
* Search for videos and display results
* User-friendly navigation
* Related videos listing
* Responsive design
* Video detail page

🎵 Spotify Clone

This project aims to create a clone of the popular music solution platform Spotify. I designed a dynamic and modern user interface using HTML, CSS, SCSS and JavaScript technologies. The project delivers content based on real data with API benchmarking and different scopes are available in accordance with responsive design principles. Features
✨ API Integration: Data such as songs, artists and albums are pulled dynamically from the API.
✨ Home Page Design: A home page similar to Spotify's modern and user-friendly interface.

🎞️ Movie Site

This project is a movie website developed using HTML, CSS, and JavaScript. To enhance user experience, it includes a dark/light mode, responsive design, and a slider created with JavaScript. Features
✨ Dark/Light Mode: Users can switch between dark and light themes.
✨ Responsive Design: Developed to be compatible with all devices.
✨ JavaScript Slider: A slider for movie showcases is included.
